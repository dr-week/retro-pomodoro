# Retro Vinyl Pomodoro Timer

A minimal, retro-style Pomodoro timer built in a single HTML file using Vue 3. 
https://dr-week.github.io/retro-pomodoro/
click to view!

## Features
* **Spinning Vinyl Record:** The record spins when the timer plays.
* **Analog Meter:** A zoomed-in analog meter smoothly pans up as time ticks down.
* **Custom Time:** Set any custom minutes before hitting play.
* **Tactile Design:** Hardware-style curved edges and responsive buttons.
* **No Build Required:** Uses Vue 3 via CDN.

## How to Use
1. Download or copy the code into an `.html` file.
2. Double-click the file to open it in any web browser.
3. Enter your desired minutes and click **SET**.
4. Click **PLAY** to start focusing.

## Tech Stack
* HTML
* CSS
* Vue.js 3 (CDN)
